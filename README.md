##### Módulo de contenido

~~~
A medida que tus aplicaciones se agrandan y tu servidor es responsable para servir más archivos, tu app.js se pondrá más complicado y difícil de leer. En lugar de tener numerosas sentencias if/else para servir diferentes contenidos de archivos estáticos, vamos a crear un módulo que servirá contenidos estáticos automáticamente. Luego, este tipo de tarea se gestiona automáticamente por un framework (el cual aprenderás en el siguiente capítulo). Toma hasta 4 horas intentando terminar esta asignación y ve cómo puedes crear tu propio módulo para hacer estas cosas. 
~~~
